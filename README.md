# 📱 IBEX Project – Android Application

## 🚀 Overview
IBEX Project is a multi-feature Android application built using Android Studio.  
It demonstrates practical implementation of modern Android development concepts including multi-activity architecture, data persistence, navigation patterns, and modular code structure.

The project was developed as part of my learning journey to strengthen real-world Android development skills.

---

## ✨ Key Features

- 🧭 Multi-screen navigation using Activities & Fragments  
- 📂 Inventory management system (CRUD operations)  
- 👤 User authentication (Sign up / Login system)  
- 📁 File handling module  
- 🏫 University listing module  
- 🗂 Navigation drawer-based UI  
- 🌐 API integration using Retrofit (if applicable)  
- 💾 Local database using Room (DAO + Entities)  
- 🌍 Arabic + English language support  

---

## 🏗️ Project Architecture

The project follows a layered structure:

- **Activities** → UI screens and app flow control  
- **Fragments** → Modular UI components  
- **Adapters** → RecyclerView data binding  
- **Data Layer**  
  - DAO (Data Access Objects)  
  - Entities (Database models)  
  - Room Database  
- **Network Layer**  
  - Retrofit API client (if used)  

---

## 🛠️ Tech Stack

- Android Studio
